Translates from German into English as “Sport Place”.

The website was created as an assignment assignment in which we were to create any sort of website that would prove useful for Refugees living in Bremen, Germany.

My Group decided to build a Website that would assist someone in finding a sporting location, with a focus on free and open offers. Data management was done with Google Sheets, in which all locations were registered. Entries would show a location's availability, sport, location and more.

We used the Google Maps API together with some custom HTML, CSS and JavaScript.

The Sport Ort Website's Main Landing Page. The Map shows the German city Bremen and all the Sporting locations entered into the system.
